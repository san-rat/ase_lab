## 📖 Overview

This project implements core e-commerce features (Requirements A through F) following a strict TDD approach. The focus areas include:
- Product & Catalog Management
- Shopping Cart functionality
- Inventory Reservation systems
- Dynamic Discounting Rules
- Secure Checkout & Payment processing
- Order Persistence & Tracking

---

## 🚀 Running the Test Suite

The entire development process was driven by tests. To execute the test suite and verify the functionality:

1. Open your terminal in the root directory.
2. Run the following command:

```bash
python3 -m pytest -q
```

---

## 📁 Repository Structure

The project is logically separated to isolate tests from production code:

- 📂 `src/` — Contains all the core domain implementation and business logic.
- 📂 `tests/` — Houses the test suite that validates the implementation and drives the TDD workflow.
